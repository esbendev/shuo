Sentence Structures

Basic Sentence Structures

Structure Example
S. + V. + O. 我喜欢你
S. + 不 + V. + O. 我不喜欢你
S. + V. + O. + 吗？ 你喜欢她吗？
S. + degree + Adj. 我很高兴。

Time and Sequence

Structure Example
when + who + prep + where + do what 明天我去商店买东西。
action + 以前 (before the action) 睡觉以前...
action + 以后 (after the action) 吃饭以后...
action1 + 以后我就 + action2 我吃饭以后我就去睡觉。

Casual or Emphasized Actions

Structure Example
repeat verb gives casual vibe 我看看书。
完 + V. （+O.） 我吃完饭了。
没有 + V. + 完 我没有吃完饭。

Collaboration and Requests

Structure Example
和 + sb. + do sth 她和中国人做生意。
请 + sb + do sth. 请你喝咖啡。
和 + sb. + 一起 我和朋友一起去公园。
