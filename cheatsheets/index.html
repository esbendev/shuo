Sentence Structures

L1-4 Foundations

Structure Example
S + V + O 我喜欢中文。
S + 不 + V + O 我不喝酒。
S + V + O + 吗？ 你是老师吗？
S + 很 + Adj. 这个菜很好吃。
S + 在 + Place 我在学校。
S + 有 / 没有 + O 我没有钱。

L5-9 Daily Conversation

Structure Example
这 / 那 + Measure word + Noun 这只狗很大。
几 + Measure word + Noun？ 你有几只猫？
谁 + 的 + Noun？ 这是谁的手机？
A + 还是 + B？ 你吃包子还是饺子？
请 + Verb + O 请给我一杯水。

L10-14 Time and Narration

Structure Example
Time + Subject + Action 明天我去商店买东西。
Action + 以前 / 以后 吃饭以后我去学校。
先 + Action1, 然后 + Action2 我先洗澡，然后睡觉。
Action1 + 以后 + 就 + Action2 我吃完饭以后就看电影。
从 + A + 到 + B 我从家到公司。

L15-19 Nuanced Use

Structure Example
V + 过 (past experience) 我去过长城。
有点儿 + Adj. 这个有点儿贵。
比较 + Adj. 这件比较好看。
又...又... 这个又便宜又好看。
虽然...但是... 虽然很贵，但是我喜欢。
不但...而且... 他不但高而且帅。
和 + sb + 一起 + do sth 我和朋友一起去饭店。
给 + sb + sth 请给我一支笔。
让 + sb + do sth 请让他告诉我。
打算 + do sth 我打算明天去商场。

Measure Words

L1-4 Core

Word Usage
个 general: 一个学生, 一个人
本 books: 一本书, 一本词典
张 flat objects: 一张纸, 一张床

L5-9 Expansion

Word Usage
只 animals / one of a pair: 一只狗
支 long thin objects: 一支笔
台 machines: 一台电脑, 一台电视
辆 vehicles: 一辆车, 三辆车
件 clothes: 一件衣服
条 long narrow items: 一条路

L10-14 and L15-19

Word Usage
口 family members: 我家有五口人
家 business / institution: 一家公司, 一家饭店
次 frequency: 一次, 两次
小时 / 分钟 duration: 一个小时, 五分钟

Quick Pattern

Word Usage
Number + Measure word + Noun 三辆车 / 两个学生 / 一支笔

Connector Meaning

也 also

L10-14 Sequence

Connector Meaning

L15-19 Nuance

Connector Meaning
但是 / 不过 but / however

不但...而且... not only...but also...
因为 because
